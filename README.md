# Demand Forecasting for Retail Planning

Forecast daily sales to improve inventory and staffing decisions.

## Aims and Objectives
- Build an end-to-end AI/ML workflow on a real-world dataset.
- Produce actionable insights and a reproducible notebook.
- Compare at least two models and explain business impact.

## Dataset
- Name: Store Sales Time Series Forecasting
- Source: https://www.kaggle.com/competitions/store-sales-time-series-forecasting/data

## Methodology
1. Data loading and quality checks
2. EDA with complementary-color visualizations
3. Preprocessing + feature engineering
4. Model training and comparison
5. Results, insights, and conclusion

## Key Insights and Results
- Baseline and improved model metrics documented in notebook.
- Feature-level interpretation and practical recommendations included.

## Tech Stack
- Python
- pandas, numpy
- matplotlib, seaborn
- scikit-learn

## How to Run
1. Install dependencies
2. Open `analysis.ipynb`
3. Run all cells sequentially
